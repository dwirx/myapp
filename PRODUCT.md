# Product

## Register

product

## Users

Developers and makers who build small HTML/React tools and want to showcase or quickly access them in one place. They work at night, run `localhost` in their browser, and prefer dark, information-dense UIs. They are the author of their tools, not end-users.

## Product Purpose

Toolsx is a personal tool gallery — a single website that collects HTML and React (TSX/JSX) mini-tools. Each tool is a file or folder dropped into `src/tools/`. The site auto-discovers tools via `tools.index.ts` and renders them: HTML tools via iframe, React tools as live components. Success means: drop a file, add one line to the index, and the tool is immediately browsable.

## Brand Personality

Precise, hacker-native, frictionless. Three words: **focused, electric, minimal**.

## Anti-references

- Notion-style soft-white with calming rounded cards — too soft
- Colorful SaaS dashboards with rainbow stat cards — too corporate
- Dribbble-shot-style glassmorphism with neon gradients everywhere — too decorative
- Generic developer-tool dark themes that feel like a VS Code extension page

## Design Principles

1. **The tool is the hero** — the chrome around tools recedes; the render is what matters
2. **Friction zero** — adding a tool should feel instant; the UI should never make you hunt
3. **Dark, not heavy** — dark surface as precision instrument, not as aesthetic choice for its own sake
4. **Density without clutter** — show many tools at a glance without visual noise
5. **Show, don't announce** — the preview thumbnail IS the label; no need for icon soup

## Accessibility & Inclusion

WCAG AA minimum. Keyboard-navigable. Reduced motion respected.
